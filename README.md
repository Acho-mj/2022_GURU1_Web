### :star: 서울여대 구루1 - python/html반

<br>

해당 파일을 다운로드 받은 후에, 가상환경 생성 뒤,
myproject > manage.py 에서 우클릭하여 터미널을 켭니다.
```
pip install django
```
django를 설치하고,
```
python manage.py runserver
```
를 통해 서버를 실행시키면 됩니다.
